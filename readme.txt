Ensure your SSL certificate is valid and your server is configured for best security practices.

== Description ==

This plugin uses the free online server test tool provided by Qualys SSL Labs to analyse your SSL certificate and server configuration are adhering to best security practices. These scans can be scheduled on a daily, weekly or monthly basis.

Notification emails can be sent to a specified email address when the grading of the test is below the configurable threshold.

== Installation ==


1. Upload the `ssl_labs` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
